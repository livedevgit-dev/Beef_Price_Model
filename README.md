# 🐂 소고기 시세 예측 모델 (Beef Price Model)

## 📂 파일 명명 규칙 (Naming Convention)

### 1. 역할 (Prefix)
- `crawl`: 외부 데이터 수집 (크롤링, API)
- `proc`: 데이터 전처리 및 마스터 생성 (Process)
- `anal`: 분석 및 모델 학습 (Analysis)
- `viz`: 시각화 및 대시보드 (Visualization)

### 2. 대상 (Category)
- `imp`: 수입 소고기 (Imported)
- `han`: 한우 (Hanwoo)
- `com`: 공통 지표 (환율, 금리 등)
